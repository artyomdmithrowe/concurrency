Файлы вынесены из конспекта `../Сокеты и TCP-сервер.md`.

## Что внутри

| Папка | Смысл |
| --- | --- |
| `01_socket_types` | Мини-заготовка по TCP/UDP/Unix-сокетам |
| `02_byte_order` | Отдельный пример вывода `htons` и `ntohs` |
| `03_minimal_tcp_server` | Минимальный однопоточный TCP-сервер |
| `04_echo_server` | Улучшенный echo-сервер с `write_all` |

## Общая проверка

```bash
cd "Билеты/Билет 1. Сокеты и TCP-сервер/code/03_minimal_tcp_server"
./script.sh
```

После запуска сервера в другом терминале:

```bash
nc localhost 8080
```
