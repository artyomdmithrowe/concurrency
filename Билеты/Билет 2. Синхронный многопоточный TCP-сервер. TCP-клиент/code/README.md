# Кодовые заготовки к билету 2

Файлы вынесены из конспекта `../Билет 2. Синхронный многопоточный TCP-сервер. TCP-клиент.md`.

## Что внутри

| Папка | Смысл |
| --- | --- |
| `01_thread_lifetime` | Мини-примеры про `join`, `detach` и опасный захват fd по ссылке |
| `02_thread_per_client_server` | Синхронный многопоточный echo-сервер: один поток на клиента |
| `03_minimal_client` | Минимальный TCP-клиент: подключиться, отправить одно сообщение, прочитать ответ |
| `04_stdin_client` | Клиент с вводом строк из stdin и `write_all` |
| `05_commands` | Команды для сборки, запуска и просмотра потоков |

## Быстрая проверка

Терминал 1:

```bash
cd "Билеты/Билет 2. Синхронный многопоточный TCP-сервер. TCP-клиент/code/02_thread_per_client_server"
./script.sh
./thread_server
```

Терминал 2:

```bash
cd "Билеты/Билет 2. Синхронный многопоточный TCP-сервер. TCP-клиент/code/04_stdin_client"
./script.sh
./client
```

Можно также подключаться через `nc localhost 11000` в нескольких терминалах.
